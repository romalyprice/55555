# Weather Diary

Автор: Попов Роман

Описание:
Программа "Weather Diary" позволяет пользователю вводить записи о погоде, фильтровать их по дате и температуре, сохранять в файл JSON и загружать из него. Это удобный инструмент для ведения дневника погоды.

Как запустить:
1. Склонируйте репозиторий.
2. Убедитесь, что установлен Python 3.
3. Запустите файл `main.py` командой:
   python main.py
4. В интерфейсе можно добавлять записи, фильтровать, сохранять и загружать данные.

Примеры использования:
- Вводите дату в формате YYYY-MM-DD.
- Вводите температуру как число.
- В описании погоды указываете краткое описание задачи или состояния.
- Для фильтрации можно указать конкретную дату или температуру выше определенной цифры.
